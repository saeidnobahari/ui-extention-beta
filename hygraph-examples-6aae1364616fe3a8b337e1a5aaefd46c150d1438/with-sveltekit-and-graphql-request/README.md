## How to Use

### Download Manually

```bash
npx degit hygraph/hygraph-examples/with-sveltekit-and-graphql-request with-sveltekit-and-graphql-request
```

Install & Run:

```bash
cd with-sveltekit-and-graphql-request
npm install
npm run dev -- --open
```
