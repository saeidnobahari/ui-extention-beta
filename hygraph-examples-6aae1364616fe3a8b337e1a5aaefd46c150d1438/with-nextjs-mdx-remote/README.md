## How to Use

### Download Manually

```bash
npx degit hygraph/hygraph-examples/with-nextjs-mdx-remote with-nextjs-mdx-remote
```

Install & Run:

```bash
cd with-nextjs-mdx-remote
npm install
npm run dev
# or
cd with-nextjs-mdx-remote
yarn
yarn dev
```
