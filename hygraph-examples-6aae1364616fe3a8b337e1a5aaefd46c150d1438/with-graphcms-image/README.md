## How to Use

### Download Manually

```bash
npx degit hygraph/hygraph-examples/with-graphcms-image with-graphcms-image
```

Install & Run:

```bash
cd with-graphcms-image
npm install
npm run dev
# or
cd with-graphcms-image
yarn
yarn dev
```
