## How to Use

### Download Manually

```bash
npx degit hygraph/hygraph-examples/with-gatsby-image with-gatsby-image
```

Install & Run:

```bash
cd with-gatsby-image
npm install
npm run dev
# or
cd with-gatsby-image
yarn
yarn dev
```
